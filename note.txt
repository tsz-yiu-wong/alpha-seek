Meme Coin AI分析与自动化平台

数据获取：利用 Python + asyncio 实现异步 HTTP 请求，保证高并发和低延迟。

数据存储：Redis缓存/实时数据处理+InfluxDB历史数据存储。
将最新的数据存入 Redis，支持快速查询和短期数据保存（例如几分钟到几小时的数据）
InfluxDB 针对时间序列数据的查询和存储进行了优化，适合存放大量历史数据，同时支持简单的聚合分析

后端API：FastAPI(python)性能高、易于开发、同时支持 RESTful API 与 WebSocket。能够高效地处理异步任务和高并发请求。

实时通信：利用 FastAPI 内置支持的 WebSocket 推送实时数据到前端，减少轮询带来的延迟。

前端框架：vue.js 上手简单，适合个人项目, 配合轻量级图表库（例如 Chart.js 或 ECharts）展示实时数据和历史走势。

服务器：选择 VPS 或云服务器（如 DigitalOcean、Linode、AWS Lightsail）即可。

(容器化：可以考虑 Docker 化各个组件（如后端、Redis、InfluxDB），方便本地开发和部署。)



1. pip install -r requirements.txt (backend)
2. npm install (frontend)
3. npm run server
